Welcome to the lit-learn documentation!
========================================

lit-learn is a toolkit dedicated to multi-task and multi-objective optimization. This documentation provides an overview of the toolkit, its modules, and how to use them effectively.

Getting Started
---------------
To get started with lit-learn, please refer to the installation instructions and basic usage examples provided in the README.md file.

Modules
-------
The lit-learn toolkit consists of several modules:

- **Core**: Contains the main optimization classes.
- **Algorithms**: Implements various optimization strategies.
- **Utils**: Provides utility functions for data handling and visualization.

For detailed API documentation, please refer to the `api` section.
